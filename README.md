# The Vanilla Pod Café Website

A warm, vintage, and cosy website for The Vanilla Pod Café, featuring a beautiful aesthetic inspired by the café's exterior and artisan food photography.

## Features

- **Fully Responsive Design** - Works beautifully on all devices
- **Warm Vintage Aesthetic** - Burgundy, cream, and tan color palette
- **Food Photography Showcase** - Gallery featuring artisan brunch photography
- **Modern Tech Stack** - Next.js 15 with TailwindCSS
- **Accessible** - Proper contrast ratios and semantic HTML

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- TailwindCSS
- Next.js Image Optimization

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
vanillapod/
├── app/
│   ├── layout.tsx      # Root layout
│   ├── page.tsx        # Main page with all sections
│   └── globals.css     # Global styles and Tailwind
├── components/
│   ├── Hero.tsx        # Hero section component
│   ├── Section.tsx     # Reusable section wrapper
│   ├── MenuCards.tsx   # Menu highlights cards
│   ├── Gallery.tsx    # Food gallery grid
│   └── Footer.tsx      # Footer component
└── tailwind.config.js  # Tailwind configuration with custom colors
```

## Color Palette

- **Burgundy**: `#6A1827` - Primary accent
- **Cream**: `#F7F1E8` - Main background
- **Warm Tan**: `#D9C7A6` - Section backgrounds
- **Charcoal**: `#2E2A28` - Text
- **Soft Gold**: `#C9A86A` - Subtle accents
